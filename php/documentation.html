Introduction

Welcome to the Employee Schedule Tool Documentation. This comprehensive guide will help you understand and utilize all features of our scheduling system effectively. Our tool is designed with simplicity and privacy in mind, requiring no registration while providing powerful scheduling capabilities.

Key Features

Privacy First

No registration or personal data collection required. All data stays in your browser.

Automatic Hour Calculation

Automatically calculates working hours, including split shifts (e.g., 12:00-16:30/17:00-20:00).

Import/Export

Support for XLSX imports and exports to PDF, PNG, and XLSX formats.

Customization

Multiple themes and customizable fonts to match your preferences.

Getting Started

Quick Start Guide

  1. Open the schedule tool in your web browser
  2. Click "Add Employee" to start creating your schedule
  3. Enter shift times in format: 09:00-17:00 or 09:00-13:00/14:00-18:00 for split shifts
  4. Use the save button regularly to preserve your changes

Detailed Guide

Working Hours Calculation

The system automatically calculates working hours in various formats:

  • Regular Shifts: Enter times like "09:00-17:00"
  • Split Shifts: Enter times like "09:00-13:00/14:00-18:00"
  • Multiple Splits: Use forward slash (/) between shifts
  • Break Handling: Breaks between split shifts are automatically excluded

Importing Data from Excel

Follow these steps to import your Excel schedule:

  1. Click "Import Data from Excel"
  2. Select your XLSX file
  3. Use the mapping interface to match columns
  4. Verify the imported data

Managing Schedules

  • Adding Employees: Click "Add Employee" and enter details
  • Split Shifts: Use the format "09:00-13:00/14:00-18:00"
  • Combining Rows: Use the combine icon for employees with multiple shifts
  • Drag and Drop: Rearrange employees by dragging rows
  • Quick Search: Use the search box to filter employees

Customization Options

  • Themes: Choose from multiple pre-designed themes
  • Fonts: Select different fonts for better readability
  • Layout: Resize columns and rows as needed
  • Print Layout: Optimized format for printing

Frequently Asked Questions

How are split shifts calculated?
Split shifts are calculated automatically when you use the format "09:00-13:00/14:00-18:00". The system adds up all working periods and excludes breaks.
Can I import my existing schedule?
Yes, you can import schedules from XLSX files. The system includes a mapping tool to match your columns correctly.
How do I save my changes?
Changes are saved automatically to your browser's local storage. You can also click the "Save Changes" button for manual saves.
What happens if I clear my browser data?
Clearing browser data will remove your saved schedule. Export your schedule to XLSX before clearing browser data.

Privacy & Security

Local Storage Only: All data is stored in your browser only.

No Data Collection: We don't collect or track any personal information.

No Server Storage: No data is sent to external servers.

Data Backup: Regular exports are recommended for data backup.
